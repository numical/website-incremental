Create a New Contract

Activity required : project.rw
Plug-in required : Project Management

This will create a new project at Awaiting Work stage so that work can be started without going through estimating and customer acceptance stages. If these are required it is better to create a new enquiry. Note that estimates can still be added to the project even if it is created as a contract.

  1. Switch to the Project Management plug-in;
  2. Click the 'New Contract' button at the top of the projects list (this is the second of the two new project buttons):

  3. A wizard will open that guides you through the rest of the creation process.
  4. The wizard will allow you to select an existing customer and product or create new ones.
  5. A project code will either be generated automatically or you will have to enter it yourself, dependent on your code settings.
  6. If you are using multiple currencies, the wizard will also ask for a base currency and an exchange rate for the contract.
