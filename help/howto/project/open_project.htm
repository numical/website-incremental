Open a Project Editor

Activity required : project.ro
Plug-in required : Project Management

  1. Switch to the Project Management plug-in;
  2. The project list will appear down the left-hand side.
  3. Double-click the project you wish to open.
  4. Note that project editors are opened read-only - you cannot edit the project until you open the read-write editor.
  5. If the required project is not visible, check that a filter has not been applied to the projects list:
