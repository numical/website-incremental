Change a Project's Status

Activity required : project.rw
Plug-in required : Project Management

  1. Open the editor for the project and switch to the read-write editor.
  2. Click the change status button on top-right of the editor.

  3. This opens a single-window wizard that offers all possible next statuses:

  4. Select the desired status, add a date then click 'Finish'. The editor will be closed and, if the option is selected. reopened.
  5. The wizard also offers any additional actions that can be carried out at the same time as the status is changed. If any of these options are selected and require additional configuration, further wizard windows will open on clicking 'Finish'.
