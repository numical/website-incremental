Open a Gantt Chart

Activity required : project.ro / marketing.ro
Plug-in required : Project Management / Marketing, Chart

Note : this is to open project-level gantt charts which display projects as the listed items. To generate a task-level gantt chart for a project you will need to export the project's tasks.

A gantt chart of selected projects can be opened either from the Project Management plug-in or from the Marketing plug-in. Each opened Gantt chart has different formats.

 

Project Management Plug-in

  1. Switch to the Project Management plug-in;
  2. The project list will appear down the left-hand side;
  3. Select the projects you wish to include in the Gantt chart (you may need to change the filter to display all the projects you wish to include);
  4. Click the button to open the Gantt chart.

 

Marketing Plug-in

  1. Switch to the Marketing plug-in;
  2. Search for the customers or products that you wish to view;
  3. Select the customers or products you wish to include in the Gantt chart;
  4. Click the button to open the Gantt chart.
  5. The Gantt chart will display all the projects for the selected customers or products.

 

Different Formats

You can switch between different formats for each opened chart using the dropdown menu button at the top:

The number of available formats will depend on the plug-ins you have and whether you have a custom plug-in contributing further formats.
