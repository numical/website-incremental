Configure Task Alerts

Activity required : project.ro, task.ro
Plug-in required : Project Management

Task alerts are controlled by a user with administrative priviledges. This administrator sets when alerts are run and configures individual alerts where approriate.

If on-going alerts are allowed, normal users can then change which ones are run within their project editors - see using layouts.

  1. Open your user's preferences;
  2. select 'System Settings/ Task / Alerts'':

  3. The window will display:
    • all available task alerts (there may be more available than shown here)
    • whether or not task alerts are run
    • which task alerts are run in which situation.
