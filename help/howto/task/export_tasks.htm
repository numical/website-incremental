Export Tasks

Activity required : project.ro, task.ro
Plug-in required : Task, Document

You export tasks to create an document of your choice. Various types of document are available by default, with additional ones you require contributed by custom plug-ins. The standard options include:

  • invoices (Microsoft Word / Excel);
  • work orders (Microsoft Word);
  • work lists (Microsoft Word).

If the status of a project is 'Awaiting Work' or beyond, the project editor will display all tasks for the project in a separate tab.

This tab will show a table of all the tasks. Select the tasks you wish to export and click the 'export' button:

Clicking this button will open a wizard that will ask for what type of document to create and any further options specific to that document type:
