Using Layouts

Activity required : project.ro, task.ro
Plug-in required : Project, Task

A layout is a recorded GUI configuration of the tasks tab of the project editor. Layouts can be used to control:

  • the displayed tasks - by selecting filters;
  • the displayed columns;
  • the detail tab display;
  • what task alerts are active.

Specific layouts can be added, saved and removed. Note that some GUI configuration is not managed via layouts.

You can select the layout to apply using the dropdown of available layouts in the top right of the tasks tab:

To change a layout, or to manage the available layouts, select the "edit..." option. This opens the layout configuration window. Each tab within this window allows a different aspect of the layout to be managed:

Detail Tabs

This tab allow you to choose if and where to display the details tabs for individual tasks.

Note that you can also choose which details tabs to display, but this has to be changed via your preferences - see 'Non-Layout GUI Configuration'.

Task Alerts

This allows to select which on-going task alerts Epic should apply to each task. Note that this might not appear if on-going task alerts are switched off.
