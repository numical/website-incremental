Create a Report

Activity required : report
Plug-in required : Reports

Note that reporting is only available in multi-user versions of Epic.

This is to create a brand new report that does not exist in the reports list in the report plug-in. If it does exist, all you need do is run it.

Creating a report is technical operation that requires training. This help page merely summarises the actions involved. Please note that Incremental can create all your reports for you. However learning to create them for yourselves offers better long-term flexibility and turn-around times.

  1. Reports are created by the creation of a report design file that holds all the information Epic needs to query the database and generate the report.
  2. Report design files are created using the 'RCP Report Designer'. This is a seperate piece of open-source software that can be downloaded and used for free. It is similar to Epic as it is based on the same technology. However it is not written by Incremental.

    Relevent web addresses:

    3. http://www.eclipse.org/birt home page of the open-source project that wrote 'RCP Report Designer'
    http://download3.eclipse.org/birt/downloads where to download 'RCP Report Designer' (look for section 'RCP Report Designer')
    http://www.eclipse.org/birt/phoenix/tutorial basic tutorial for 'RCP Report Designer' - this should be tried after training from Incremental on Epic settings and concepts

  3. Once a report is completed in the RCP Report Designer, it is saved as a report design file. This is then moved to a specific directory on the server component of Epic (the precise location depends on your set-up).
