Run a Report

Activity required : report
Plug-in required : Reports

Note that reporting is only available im multi-user versions of Epic.

This is to run a report that is already available from the reports list in the report plug-in. If the report you want does not exist you must first create it.

  1. Switch to the Reports plug-in;
  2. Select the reports from the reports list (it may be in the 'recently viewed' category):

  3. Double-click to run the report. If any parameters are required these will be prompted for:

  4. Press finish and wait for the report to be run - this could take a few seconds, depending on the complexity of the report.You can do other things during this time, including starting other reports:

  5. When the report is displayed you can view it full screen or save or print via the buttons at the top. If the report has parameters you can change these and re-run the report via the button at the bottom:
