Add an Exchange Rate

Activity required : admin
Plug-in required : Any

Exchange rates are only used if Epic is using multiple currencies.

  1. Switch to the administration plug-in.
  2. Expand the 'Currency' section.
  3. Open the 'Exchange Rates' editor by double-clicking on 'Add Exchange Rate(s)'.
  4. Use the buttons to add and remove exchange rates - note you cannot edit existing exchange rates:

  5. Ensure you save the changes when you have finished (you will be prompted to do this when you close the editor).

WARNING : deleting exchange rates might prevent multi-currency records opening
