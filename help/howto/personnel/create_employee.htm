Create a New Employee

Activity required : personnel.rw
Plug-in required : Personnel

  1. Switch to the Personnel plug-in;
  2. Ensure the you are viewing the employees list and click the new employee button:

  3. This will open a single window wizard that asks for all mandatory information for the new record:

  4. An employee code will either be generated automatically or you will have to enter it yourself, dependent on your code settings.
  5. Clicking 'Finish' will open the employee editor where further details can be added. The new record is not created unit this editor is first saved.
