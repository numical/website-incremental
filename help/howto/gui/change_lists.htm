Change Dropdown Lists

Activity required : admin
Plug-in required : Administration

Epic uses dropdown lists such as the example below to provide standard values for fields:

Generally Epic is quite flexible - if you do not find the value you want in the list you can type it directly into the field instead. However this is not always the case and, moreover, you will wish to tailor the provided list to the values you most commonly use. This is done by:

  1. Switch to the administration plug-in.
  2. Expand the 'List and Dropdowns' section.
  3. Open the 'List and Dropdowns' editor by double-clicking on 'Edit':

  4. This lists all sets of values used by Epic, including categories for categorised records. Each is in a separate section that you expand to see current values.
    You can add and remove values via the buttons and directly edit the existing values.
  5. Ensure you save the changes when you have finished (you will be prompted to do this when you close the editor).
