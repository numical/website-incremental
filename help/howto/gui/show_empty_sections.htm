Show Empty Sections

Activity required : None
Plug-in required : Any

By default read-only editor do not show sections that have no data - that are 'empty'.

However you may prefer to see all the types of data that a record can contain. In which case you can choose to see all sections whether or not they have data.

  1. open your user preferences
  2. select 'Your Preferences / General'

  3. untick 'hide empty sections (read-only)'
