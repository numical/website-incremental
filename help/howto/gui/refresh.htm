Refresh Display

Activity required : None
Plug-in required : Any

There are two circumstances where Epic may not show the latest information:

  • changes in one plug-in may not be immediately reflected in other plug-ins;
  • changes are made by other users.

Wherever this may occur Epic always provides a refresh option, which will update the current list or editor with the latest information from the database. This option is indicated by a button or menu option with the icon.
