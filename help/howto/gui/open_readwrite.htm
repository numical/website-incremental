Open Read-Write Editor

Activity required : Plug-in specific read-write activity
Plug-in required : Required plug-in

Editors open read-only. To edit the record you must open the equivalent read-write editor. This is done by clicking the 'Edit' icon found in the top right of each editor:

This icon will only appear if you have the appropriate read-write activity in order to edit the record.

On clicking the icon, Epic will check if anyone else is editing the record - if so Epic will tell you who and not open the read-write editor. This prevents you from overwriting any information the other person is in the process of adding.

The read-write editor will open with same expandable sections expanded as the read-only editor.
