Change Expanded Sections

Activity required : None
Plug-in required : Any

You can define which expandable sections are automatically expanded when you open an editor via your preferences.

Note that when you switch from a read-only to a read-write editor the same sections are expanded, irrespective of the settings below.

  1. open your user preferences
  2. select 'Your Preferences / Editor Sections'

  3. select which sections you want automatically expanded
  4. ensure to click 'Apply' before clicking 'OK'
