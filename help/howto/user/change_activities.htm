Change a User's Activities

Activity required : admin
Plug-in required : Administration

Note that you cannot your own activities.

  1. Switch to the administration plug-in.
  2. Expand the 'User Management Tools' section.
  3. Open the 'Users' editor by double-clicking on 'Edit users'
  4. Select the user
  5. Select the activities to add/remove and use the '<<' and '>>' buttons.
  6. Multiple users can be edited at once.
  7. Save the changes.
