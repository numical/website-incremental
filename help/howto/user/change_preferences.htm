Change a User's Preferences

Activity required : None
Plug-in required : Any

Access your preferences via 'Edit/Preferences' off the main menu:

All available preferences will then be displayed in a tree down the left-hand side. Click a preference to see what options are available.

If you make changes, ensure you click the 'Apply' (bottom-right) before closing the window 'OK' or 'Cancel'. The 'Restore Defaults' button returns the settings for that preference to Epic's initial values.
