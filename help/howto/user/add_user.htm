Add a User

Activity required : admin
Plug-in required : Administration

  1. Switch to the administration plug-in.
  2. Expand the 'User Management Tools' section.
  3. Open the 'Users' editor by double-clicking on 'Edit users'
  4. Use the 'Add User' button.
  5. Select the activities to add using the '<<' and '>>' buttons.
  6. Multiple users can be added at once.
  7. Save the changes.
  8. A message will warn you that the user identities have to be set up on whatever system Epic uses for authentication - the system administrator will have procedures for this.
