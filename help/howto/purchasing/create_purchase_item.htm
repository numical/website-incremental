Create a New Purchase Item

You can create a purchase item in one of two ways:

  • from scratch;
  • via the project editor for a project with a status of 'Awaiting Work' or beyond.

Once the purchase item has been created, it then needs to be progressed through its own lifeline.

From Scratch

Activity required : purchase.rw, purchase.authorise
Plug-in required : Purchases

  1. Switch to the Purchase plug-in;
  2. Ensure that you are viewing the purchase items list and click the new purchase item button:

  3. A single window wizard will open allowing you to enter the required information. This includes selecting a cost center (normally a task). Epic ensures that you can only assign purchase items to valid cost centers (e.g. tasks must be authorised or started on an on-going project):

  4. The code may have to be manually entered , depending on your code settings;
  5. Once you have entered sufficient information the 'Finish' button will be enabled. Clicking this will close the wizard and open the purchase item editor allowing you to enter further information.

 

 

Via Project

Activity required : purchase.rw, purchase.authorise, task.rw
Plug-in required : Project Management, Purchases

  1. Open the parent project's editor;
  2. Open the equivalent read-write editor;
  3. Select the 'Tasks' tab for the project (if there is no such tab then the project is at an inappropriate status) and then select the task for which the new purchase item is for;
  4. Select the 'Purchases' sub-tab and then open either 'Materials' or 'Subcontracts' section within this (depending on whether you want to initiate a purchase of material or a service. The 'request new ...' button is enabled:

  5. Click this button and the new purchase item wizard opens. This time the cost center is pre-selected;
  6. Once you click 'Finish' the item is added to the list displayed in the project editor. To add more details double-click this and the project editor will open.
