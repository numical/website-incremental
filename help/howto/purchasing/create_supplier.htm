Create a New Supplier

Activity required : purchase.rw
Plug-in required : Purchases

Do you want to create a supplier or a company record? If the latter see create a new company.

A new company record can be created in one of two ways:

  • a company record can be converted;
  • directly:

This page details how to create a supplier directly

  1. Switch to the Purchase plug-in;
  2. Ensure that you are viewing the suppliers list and click the new supplier button:

  3. A supplier editor will open to allow you to add details:
    • only the supplier's name is required
    • the supplier record is not created until the editor is first saved
