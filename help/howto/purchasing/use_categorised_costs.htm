Use Categorised Costs

Activity required : purchase.ro
Plug-in required : Purchases

Epic stores up to three fields for a purchase item's cost. Normally however it simply uses and displays the first of these.

If you wish to use all three:

  1. open your user preferences
  2. select 'Your Preferences / Purchases

  3. select 'show categorised purchase costs'

You can now use all three fields - these can be defined as anything you want and their descriptions customised via a custom plug-in. However note that for all price and summation calculations, Epic adds the three values together.

This is an example of progressive disclosure.
