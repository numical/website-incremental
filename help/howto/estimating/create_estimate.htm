Create a New Estimate

Activity required : project.rw, estimating.rw
Plug-in required : Project Management, Estimating

This details how to create an estimate record - the first step to generating an estimate. See the estimate tutorial for how to generate an actual estimate once the record has been created.

Estimates are created via their parent project:

  1. Open the parent project's editor
  2. Open the equivalent read-write editor
  3. Expand the 'Estimate(s)' section and click the 'Add Estimate' button:

  4. This will start a single-window wizard:

  5. Select the estimator and whether the estimate is internal or external.
  6. The estimate code will either be generated automatically or you will have to enter it yourself, dependent on your code settings.
  7. Click 'Finish' - an estimate record is added to the project.
  8. You can now open the estimate record. See the estimate tutorial for how to generate the actual estimate.

Note that when changing a project's status from 'Enquiry' to 'Estimating' you are also given the option to add an estimate. This opens the same wizard.
