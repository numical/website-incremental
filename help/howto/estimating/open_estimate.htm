Open an Estimate Editor

Activity required : estimating.ro
Plug-in required : Estimating

There are two ways to open an estimate:

From the Estimate List:

  1. Switch to the Estimating plug-in;
  2. The estimates list will appear down the left-hand side;
  3. Double-click the estimate you wish to open;
  4. Note that estimate editors are opened read-only - you cannot edit the estimate until you open the read-write editor;
  5. If the required estimate is not visible, check that a filter has not been applied to the estimates list:

  6. If the estimate is still not visible click the refresh button to ensure Epic is displaying the latest data from the database.

 

 

From the Project Editor:

  1. Open the parent project's editor;
  2. Expand the 'Estimate(s)' section;
  3. Double-click the estimate (if read-only editor):

  4. click the 'Add Estimate' button (if read-write editor):
