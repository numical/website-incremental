Configure the Document Repository

Activity required : None
Plug-in required : Any

Various editors offer an expandable section titled 'Documents'. This lists any documents associated with the record that are stored in the document repository.

However if the following message is displayed you must configure the document repository:

Configuring the document repository consists of creating an empty folder (directory) that can be seen by all the computers on which Epic is installed and then pointing each installation of Epic at that folder:

  1. Create the empty folder. In the case of a standalone version of Epic such as the demo this can be anywhere on your computer. In the case of a multi-user version of Epic, the system administrator must set up a central, shared folder than be written to by all users of Epic;
  2. Open your user preferences;
  3. Select 'System Settings / Document Generation':

    • Temporary file directory : a working directory for Epic - it is recommended to leave the default value
    • Repository root directory : set this to the folder you created in step 1.
    • Template directory : a utility directory where all common templates can be put - see templates
  4. Set the 'Repository root directory' and click 'Test' to ensure Epic can use it:
