Create a New Customer

Activity required : shiprepair.marketing.rw
Plug-in required : Marketing

Do you want to create a customer or a company record? If the latter see create a new company.

A new customer record can be created in one of four ways:

  • a company record can be converted;
  • during the create enquiry or create contract process;
  • whilst creating a new association;
  • directly:

This page details how to create a customer directly

  1. Switch to the Marketing plug-in;
  2. Ensure the 'customers' option on the search criteria is selected and click the new customer button:

  3. A customer editor will open to allow you to add details:
    • only the customer's name is required
    • the customer record is not created until the editor is first saved
