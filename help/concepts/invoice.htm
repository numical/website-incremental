Invoice

An invoice is a priced listing of some or all of the tasks in a project.

Incremental Epic calculates the costs and prices of individual tasks automatically - see the Pricing Tutorial for full details. Selected tasks can be exported to a document to create an invoice - see invoice generation. By default Epic can create Microsoft Word and Excel invoice documents but further types can be contributed via a custom plug-in.
