Welcome Screen

The welcome screen is an optional GUI utility to help navigate around Epic. It opens automatically when Epic starts and remains visible unless explicitly closed. It offers a list of plug-ins, a list of recently viewed records, and access to Help functionality including a link to the Getting Started tutorial.
