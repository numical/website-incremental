Document

A document is any file that is read outside Epic by another application such as a word processor or a spreadsheet. Documents can be added to various records within Epic such as companies and projects. Epic can store any document you choose.

Adding a document causes a copy of it to be placed in Epic's document repository. This means that anyone using Epic can now view the document. If you wish to further edit the document, you must open it via Epic. This ensures that the repository copy is opened rather than your original file.

Documents are separate files. This means that Epic cannot search their content

Epic can also create documents as part of its export processes. Such functionality is generally provided by templates or by custom plug-ins.
