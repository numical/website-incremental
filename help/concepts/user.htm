User

A user is the identity given to anyone using Epic. This identity is tied to the username / password combination entered during login.

A user has a set of preferences that are recorded centrally so these are used on whatever computer the user uses (providing the computer has the appropriate plug-ins installed).

A user is assigned a set of activities that define what they can do (again, providing the appropriate plug-ins are installed).
