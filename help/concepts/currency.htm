Currency

By default Epic considers all costs and prices to be in your local currency - as defined by which country your computer believes it is in.

However Epic can handle multiple currencies - converting between them using exchange rates.

Once you have switched Epic to use multiple currencies, projects use a base currency and an exchange rate date to decide on the conversion rate to use (see exchange rate for more information. Other records use the latest exchange rate available.
