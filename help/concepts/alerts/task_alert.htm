Task Alert

A task alert is a some check that is run against a specific task. The result of this check is a traffic light metaphor:

red - a problem
amber - a warning
green - fine

A text reason for a red or an amber result is also available.

Task alerts can be run in two places within Epic, dependent on their configuration:

  1. When a project's status is changed from 'Work Ongoing' to 'Creating Invoice' the change project status wizard will offer you the chance to run the checks (this may be mandatory):

  2. Alerts run automatically with the tasks list of the project editor if the alert columns are made visible via the layout:

Epic has several built-in task alerts. However the real power of alers come form you defining additional ones useful to yourself through a custom plug-in.
