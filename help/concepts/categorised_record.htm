Categorised Records

Some types of record can be categorised. This means the record has up to three additional text fields that can be used to decribe or categorise that record.

Categories can be then be used in such functionality as marketing searches and report generation.

The values in category fields can by anything you want and can be set, if you have the 'admin' activity, using the 'Lists and Dropdowns' editor.

A record's categories are displayed in their own expandable section. If there are no possible category values, this section is not displayed.

The following types of record can be categorised:

  • companies
  • contacts
  • customers
  • employees
  • products
  • purchase items
  • suppliers
  • vessels
