Gantt Chart

Epic can produce Gantt charts showing dates and durations for selected projects.
Different formats are available and further ones can be added via a custom plug-in.

Note that gantt charts are at the project level - projects are the items listed. If you want a Gantt chart of tasks within a project you will need to export the tasks to an external format such as Microsoft Project and generate the Gantt chart there.

Gantt charts can be viewed in Epic, printed or saved as Adobe Acrobat PDF files.
