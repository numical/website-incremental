Template

A template is a base file from which Epic can automatically generate a document.

A template file is typically a normal Microsoft Word document file with placeholders in to allow Epic to substitute actual values. For instance it may have content such as:

"Dear [TITLE] [SURNAME],
We regret we must decline to offer to tender for the ..."

The power of templates is that you can edit them as normal documents, changing their format and content as you wish, as long as you leave the placeholders. Epic will then pick up an reflect these changes in all documents it produces.

Epic templates are similar to 'Mail Merge' functionality in programs such as Microsoft Word. However Epic templates allow much more powerful control of tables (what columns etc.) and other objects such as watermarks.
