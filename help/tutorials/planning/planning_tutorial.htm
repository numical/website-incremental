Planning Tutorial

Activity required : planning.ro, planning.rw
Plug-in required : Project Management, Planning

Incremental Epic offers two ways to generate and update project plans.
You can use the in-built functionality of Epic, or use its integratation with dedicated external planning tools.
Such a tool is Microsoft Project™ which Incremental Epic supports 'out-of-the-box'.
Other planning tools can be integrated through custom plug-ins.

There are two versions of this tutorial.
The first describes using Incremental Epic's in-built functionality, the second describes using Microsoft Project.

  • Epic's in-built functionality
  • Microsoft Project™ integration
