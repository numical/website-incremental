Epic On-line Help

This on-line help is a copy of the help file contained in version 1.5.7 of Incremental Epic. If your version of Incremental Epic is not up-to-date the contents may differ.

The on-line version does not have the search, navigation assistance and bookmarking support of Epic's in-built Help system.
